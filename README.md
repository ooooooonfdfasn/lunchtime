# 점심시간 Prototype

직장인이 점심시간에 랜덤 익명으로 들어와 메뉴를 정하고, 가볍게 소통하고, 필요하면 신고/차단할 수 있는 모바일 우선 프로토타입입니다.

## 포함된 흐름

- 스프레드시트형 첫 화면: 열/행 헤더, 비상호작용 셀 배경, 병합 셀 영역
- 상단 배너: 오른쪽에서 왼쪽으로 흐르는 정보 티커
- 테마 선택: 엑셀 안내, 블랙/화이트 모드 전환
- 셀 배경: 실제 DOM 셀로 구성하되 선택/하이라이트 없이 배경처럼 표시
- 시간 기준 전환: 24h / 12h
- 채팅 보내기: 중앙 하단 버튼에서 미니 채팅창 열기

## 실행

`index.html`을 브라우저에서 열면 바로 동작합니다. 별도 서버나 패키지 설치가 필요 없습니다.
