# (주) 백수 Prototype

집에 있지만 함께 근무중인 사람들을 위한 메인 페이지와 점심시간 익명 소통 화면을 포함한 프로토타입입니다.

## 포함된 흐름

- 메인 페이지: `(주) 백수` 타이틀, 슬로건, 공통 기능, 점심시간 입장 동선
- 스프레드시트형 첫 화면: 열/행 헤더, 비상호작용 셀 배경, 병합 셀 영역
- 상단 배너: 오른쪽에서 왼쪽으로 흐르는 정보 티커
- 셀 배경: 실제 DOM 셀로 구성하되 선택/하이라이트 없이 배경처럼 표시
- 시간 기준 전환: 24h / 12h
- 채팅 보내기: 중앙 하단 버튼에서 미니 채팅창 열기

## 실행

`index.html`을 브라우저에서 열면 메인 페이지가 뜨고, `점심시간 입장`으로 `lunchtime.html`에 접속할 수 있습니다. 별도 서버나 패키지 설치가 필요 없습니다.
